# MVP_Ship_Classification.ipynb
# Ship Classification using Deep Learning

Projeto desenvolvido como MVP da disciplina de Machine Learning.

## Objetivo

Desenvolver modelos de Deep Learning capazes de classificar automaticamente diferentes tipos de embarcações a partir de imagens.

## Dataset

Game of Deep Learning: Ship Classification Dataset

## Modelos Avaliados

- CNN Baseline
- MobileNetV2
- ResNet50

## Melhor Modelo

MobileNetV2

### Resultados

- Accuracy (Validação): 86,7%
- Accuracy (Teste): 86,3%
- F1 Macro: 0,87

## Tecnologias Utilizadas

- Python
- TensorFlow / Keras
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Google Colab
